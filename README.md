# 남양주시장애인복지관 세로 키오스크 v6

## v6 수정 내용
- 주요 일정은 뜨는데 유튜브만 안 뜨는 경우를 확인하기 쉽게 디버그 문구를 추가했습니다.
- `youtubeId`, `youtubeID`, `youtubeld`, `유튜브ID`, `유튜브영상ID`, `유튜브아이디`, `youtubeUrl`, `유튜브URL` 입력을 모두 인식합니다.
- `항목=유튜브ID`, `내용=영상ID` 방식뿐 아니라 `유튜브ID` 전용 열에 입력해도 인식합니다.
- 브라우저 캐시 때문에 예전 JS가 남아 있을 수 있어 `?v=6` 캐시 방지 값을 추가했습니다.

## 가장 중요한 확인
OneDrive xlsx를 읽고 있는지 확인하려면 주요 일정 제목을 임시로 `연결테스트123`처럼 바꾼 뒤 화면을 새로고침하세요.
바뀌면 OneDrive 연결 성공입니다. 안 바뀌면 GitHub 안의 기본 파일 또는 예시 데이터가 뜨는 상태입니다.

## 유튜브 입력법
xlsx에서 아래처럼 입력하세요.

| 구분 | 항목 | 내용 |
|---|---|---|
| 설정 | 유튜브ID | 1EM5Ykytyn4 |

또는

| 구분 | 항목 | 내용 |
|---|---|---|
| 설정 | 유튜브URL | https://youtu.be/1EM5Ykytyn4 |

## config.js
OneDrive 공유 링크를 넣습니다.

```js
window.KIOSK_CONFIG = {
  DATA_FORMAT: "xlsx",
  DATA_URL: "OneDrive xlsx 공유 링크"
};
```

## 영상이 계속 안 뜰 때
1. 주요 일정 제목을 OneDrive에서 바꿔서 화면에 반영되는지 확인
2. 반영되지 않으면 DATA_URL이 실제 OneDrive 파일이 아닐 가능성 있음
3. 반영되는데 영상만 안 뜨면 유튜브 영상의 임베드 허용 여부, 기관망의 YouTube 차단 여부 확인
